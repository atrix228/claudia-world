## 🤖 What is Claudia World?

Claudia World is a powerful AI assistant that doesn't just chat — **it acts**. With 22 built-in tools, it turns your phone into an AI-powered command center. Connect to any OpenAI-compatible API provider and use the best AI models available.

Ask your AI to *"check my notifications and summarize them"* — it will read, analyze, and brief you. That's an **agent**, not a chatbot.

## 🧠 All Major AI Models

Connect to your preferred AI provider and access any model:

- **Claude** Opus, Sonnet, Haiku (Anthropic)
- **GPT-4o**, GPT-4 Turbo, o1, o3-mini (OpenAI)
- **Gemini** Pro, Flash (Google)
- **DeepSeek** V3, R1
- **Qwen**, Llama, Mistral, Command R+
- Any model via OpenAI-compatible API

### Supported Providers

- [OpenRouter](https://openrouter.ai) — 200+ models in one place
- [Together AI](https://together.ai) — Fast, affordable inference
- Fireworks AI, Groq, Perplexity
- Self-hosted servers (Ollama, vLLM)
- Direct provider APIs (Anthropic, OpenAI)

**One app, unlimited providers.** Configure your endpoint and API key in Settings.
